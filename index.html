Mural Criativo

Uma corrente de criatividade em HTML e CSS. Cada carta é uma interpretação de um tema.

Como Participar

Siga estes 4 passos simples para adicionar sua carta à corrente.

1

Encontre o Tema

Veja o card "Próximo Tema" no final da fila para saber qual o tema atual.

2

Crie sua Arte

Use HTML e CSS para criar uma carta visual que interprete o tema proposto.

3

Passe o Tema Adiante

Dentro do seu arquivo HTML, defina qual será o próximo tema para o desenvolvedor seguinte.

4

Envie seu PR

Faça um Pull Request no GitHub com sua contribuição para adicioná-la à corrente.
